#  Crypto Website
[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge)](https://nowakandrzej283.github.io/crypto-website/#)

A modern and responsive crypto dashboard built with HTML, CSS, and JavaScript.  
Website focuses on real-time crypto market data with a clean and intuitive interface.

---

## 📌 Features

- 📈 Real-time crypto market tracking (UI simulation)
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Clean and fast user interface

---

## 🖥️ Sections

- Header
- Hero section
- About Us
- Pricing
- CTA section
- Footer

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎨 UI/UX Focus

This project focuses on modern fintech dashboard design principles:

- Clean typography
- Modern layout
- Smooth spacing and hierarchy
